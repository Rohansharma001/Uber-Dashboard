This repository contains an interactive Uber Analytics Dashboard built using Microsoft Power BI. The dashboard provides insights into ride patterns, revenue trends, trip duration, and demand behavior to support data-driven decision-making.
-->Objectives

Analyze Uber ride data to identify key business trends

Monitor performance metrics such as total rides, revenue, and average fare

Understand peak hours, high-demand locations, and trip distribution

Enable interactive and dynamic reporting using Power BI

Dashboard Features

Total rides, total revenue, and average trip metrics

Time-based analysis (daily, weekly, monthly trends)

Location-based insights using maps

Peak hour and demand analysis

Interactive slicers for date, city, and vehicle type
